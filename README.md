# Install

# usage

## Help:
`npx srtcl --help`

## To move 10second forward:
`npx srtcl <filepath> -add 10 -o <outputpath>`

## To move 10 second backward:
`npx srtcl <filepath> -remove 10 -o <outputpath>`
